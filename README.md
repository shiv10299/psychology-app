# 🧠 Psychology Insight App

An AI-powered personality assessment app that analyzes your psychological profile through a 20-question quiz and generates deep, personalized insights using Claude AI.

---

## ✨ Features

| Feature | Description |
|---|---|
| **20-Question Quiz** | Psychologically designed questions targeting 5 core traits |
| **Scoring Engine** | Per-question trait mapping (no generic A/B/C logic) |
| **8 Personality Types** | Custom archetypes: Silent Analyzer, Emotional Strategist, etc. |
| **AI Insight** | Deep, personalized analysis generated by Claude AI |
| **AI Chat** | Talk to a psychological AI assistant informed by your profile |
| **Dark Luxury UI** | Animated, editorial-style interface with smooth transitions |
| **Keyboard Navigation** | Press A/B/C to select, Enter to advance, Backspace to go back |

---

## 🧱 Tech Stack

**Frontend:** React 18, React Router v6, Axios, CSS Variables  
**Backend:** Node.js, Express.js, Anthropic SDK  
**AI:** Claude (claude-sonnet-4) via Anthropic API

## 🚀 Setup & Running Locally

### Prerequisites

- Node.js v18+
- npm v8+
- An [Anthropic API key](https://console.anthropic.com/)

---

### Step 1: Clone and navigate

```bash
cd psychology-app
```

---

### Step 2: Set up the Backend

```bash
cd backend

# Install dependencies
npm install

# Create your .env file
cp .env.example .env
```

Open `backend/.env` and add your Anthropic API key:

```env
PORT=5000
ANTHROPIC_API_KEY=sk-ant-your-key-here
NODE_ENV=development
FRONTEND_URL=http://localhost:3000
```

Start the backend:

```bash
# Development (with auto-reload)
npm run dev

# OR production
npm start
```

You should see:
```
🧠 Psychology Insight API running on port 5000
📡 Health: http://localhost:5000/api/health
🔑 AI Key: Configured ✓
```

---

### Step 3: Set up the Frontend

Open a **new terminal** tab:

```bash
cd frontend

# Install dependencies
npm install

# Start the React dev server
npm start
```

The app opens at **http://localhost:3000**

> The frontend proxies `/api` requests to `http://localhost:5000` automatically (configured in `package.json`).

## 🔌 API Endpoints

### POST `/api/insight/analyze`
Submit quiz answers and receive personality analysis + AI insight.

**Request:**
```json
{
  "answers": ["A", "C", "B", "A", "B", "C", "A", "B", "C", "A", "B", "A", "C", "B", "A", "C", "B", "A", "C", "A"]
}
```

**Response:**
```json
{
  "success": true,
  "result": {
    "scores": {
      "overthinking": 72,
      "emotional": 58,
      "logic": 45,
      "social_energy": 30,
      "confidence": 40
    },
    "personalityType": "Deep Overthinker",
    "typeDescription": "A profound mind that processes everything deeply...",
    "aiInsight": "There's a particular kind of exhaustion that only you know...",
    "generatedAt": "2025-01-01T12:00:00.000Z"
  }
}
```

---

### POST `/api/chat/message`
Send a chat message and receive a psychological AI response.

**Request:**
```json
{
  "message": "Why do I keep overthinking everything?",
  "sessionId": "uuid-here",
  "userProfile": {
    "personalityType": "Deep Overthinker",
    "scores": { "overthinking": 72, ... }
  }
}
```

**Response:**
```json
{
  "success": true,
  "response": "Overthinking often isn't random — it's a form of...",
  "sessionId": "uuid-here",
  "messageCount": 1
}
```

---

### DELETE `/api/chat/session/:sessionId`
Clear a chat session's history.

## 🔒 Production Considerations

- Replace in-memory chat session store with **Redis** or **MongoDB**
- Add **authentication** (JWT) for persistent user profiles
- Store quiz results in **MongoDB** for history and analytics
- Add **input sanitization** (e.g., `express-validator`)
- Set `NODE_ENV=production` to hide internal error details
- Use a **reverse proxy** (nginx) in front of Express

## 🛠 Troubleshooting

**"AI Key: Missing"** — Check that `ANTHROPIC_API_KEY` is set in `backend/.env`

**CORS errors** — Ensure `FRONTEND_URL` in `.env` matches your React dev server URL

**"Expected 20 answers"** — The quiz must be completed fully before submitting

**AI insight is null** — The result page still shows trait scores; the AI call failed silently. Check your API key and quota.

**Port conflict** — Change `PORT` in `.env` and update the proxy in `frontend/package.json`
